Últimos Projetos

Check out some of my latest projects with creative ideas.

Gerador de Propostas

Aplicação web desenvolvida para automatizar a criação, personalização e envio de propostas comerciais profissionais. O sistema permite que empresas e prestadores de serviço criem propostas em PDF com identidade visual própria, detalhes de serviços/produtos, valores, formas de pagamento, validade da proposta e assinatura digital.

Encurtador de Links

Sistema web para encurtamento de URLs com geração de links curtos, estatísticas em tempo real e foco em segurança, desempenho e rastreamento inteligente. Ideal para empresas, afiliados e criadores que desejam acompanhar o desempenho de links com precisão.

Calculadora Freelancer

Aplicação web voltada para freelancers e profissionais autônomos calcularem o valor ideal de seus serviços com base em horas de trabalho, despesas fixas, impostos e lucro desejado. O sistema ajuda na formação de preços de forma profissional, transparente e sustentável.

Baixador de Vídeos do YouTube

Aplicação web para baixar vídeos e áudios do YouTube de forma rápida e simples. O sistema permite inserir a URL do vídeo e escolher o formato desejado (MP4, MP3, resoluções variadas), com foco em usabilidade, desempenho e compatibilidade com múltiplos dispositivos.

Gerador de README.md para GitHub

Aplicação web que facilita a criação de arquivos README.md personalizados para projetos no GitHub. O sistema guia o usuário por etapas simples para gerar um README estruturado, com foco em clareza, boas práticas e apresentação profissional do repositório.

Baixador de Thumbnails do YouTube

Ferramenta web simples e rápida para baixar a imagem de capa (thumbnail) de qualquer vídeo do YouTube em diferentes resoluções. Basta inserir a URL do vídeo e escolher entre os tamanhos disponíveis (HD, SD, MQ).

Minificador de Código CSS

Ferramenta online para minificar arquivos CSS, reduzindo o tamanho do código ao remover espaços, quebras de linha e comentários, sem afetar o funcionamento do estilo. O objetivo é entregar CSS otimizado para produção, melhorando performance de carregamento e reduzindo o consumo de banda.

Criador de QR Code e Links PIX

Aplicativo web que gera códigos PIX válidos, incluindo link de pagamento e QR Code, prontos para serem usados em transações reais. Permite que usuários criem cobranças instantâneas para valores fixos ou variáveis, com dados completos para receber pagamentos via PIX.
